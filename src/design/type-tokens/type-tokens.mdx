<h1 className="h1">Typescale, Tokens, and Mixins<img src={Comments} alt="Text" /></h1>

<span className="p">
  The Easel design system's default type scale is based on the Major-Third approach, measured in `rem`. Each `rem` being
  equal to `16px`. In typography, a major third refers to a difference in font size of `1.250rem`.
</span>
<span className="p">
We have 3
Classifications of Text Styling: Display, Heading and Body (Labels are a sub-set of Body). Each classification has a
set of tokens that are used to style the text. These tokens are used in conjunction with our Mixins to create a
consistent and scalable design system.

</span>

### <span className="h2">Using the tokens</span>

Use the above tokens along with the `text` mixin to apply the display text styling to an element. The text mixin is found in the `_type.scss` partial file.

<Source code={`
.pah-card__title {
  @include text($display0);
}

`} dark language="scss" />
