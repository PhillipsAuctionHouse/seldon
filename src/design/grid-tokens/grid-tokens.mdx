## <span className="h2">Grid Mixins</span>

<span className="p">
  There are two mixins that are used to create the grid system. The first is the `gridContainer` mixin which is used to implement the grid system outlined above. There is an optional argument that you can pass this mixin to remove the default margins set on all grids. The [`Grid` component](?path=/story/components-grid--playground), which utilizes this mixin, should be used to layout your sections.

<Source code={`@include gridContainer;`} dark language="scss" />

</span>

<span className="p">
  The other is a general grid mixin helper. You can use this mixin to define any grid you want by passing in the number of columns you want to span. When applied to an element this mixin turns the display to grid, sets the grid template columns to the number of columns you want to span, and sets the gap between the columns to the gutter size.

<Source code={`
@include grid(6);

// This will return the following styles

display: grid;
grid-template-columns: repeat(6, 1fr);
gap: $spacing-medium;

`} dark language="scss" />

</span>
